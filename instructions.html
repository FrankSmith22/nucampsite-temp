In order to set up a bootstrap project:

    This is assuming you have node/NPM installed

  1. Create a folder somewhere and open it in VSCode
  2. Create an "index.html"
  3. Open a terminal and run "npm init"
  4. accept all defaults except for "main", where you should type "index.html"
    5. npm and node have now been configured for your project
  5. run "npm install lite-server --save-dev" to install a server for hot-reloading your code during dev
  6. Add these two lines to the scripts object in package.json:
    7. "start": "npm run lite",
     "lite": "lite-server",
    lite-server has now been configured for your project
  7. run "npm install bootstrap@4.5.2" and "npm install jquery@3.5.1 popper.js@1.16.1" to install bootstrap and jquery
  8. Every page must have a head that references the proper scripts,
    as well as the js files before the closing body tag,
    just like in your nucampsite html files
Your project is now fully ready to utilize bootstrap
